The Making of This Website

How I created my personal website, and some of the resources I used

Why I Started Building This Site

My personal website was originally created as part of an activity for CSE199: How the Internet Works. This is how I learned about most of the resources and tools I've listed below. For my final project, I decided to expand on what I'd started and create a functional personal website that could be used professionally.

Resources and Tools

GitHub Pages
This site is hosted from the emilo0o.github.io repository using GitHub pages. To make, commit, and push changes I used a both a terminal and the GitHub app for Windows.
Atom
This is the editor I used for the website code, which works well with GitHub. It's free to download and use.
W3 Schools
I used W3 Schools' tutorials to learn more about HTML, CSS and Bootstrap. I also used one of their bootstrap templates as a starting point.
Bootstrap
My site gets its styling from a CDN link to Bootstrap, a popular web design framework.
Page Speed Insights
Mobile-Friendly Test
These resources from Google are useful for testing how quickly a page loads and whether the page is mobile friendly. Another way to check for mobile responsiveness that I found useful is to use the mobile device view in the chrome developer tools.
